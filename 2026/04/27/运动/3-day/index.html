💪 哑铃三分化训练手册（家庭减脂优化版）

💪 哑铃三分化训练手册（家庭版）

🎯 目标：减脂 + 保肌肉 + 出线条
🏠 场景：居家训练（哑铃 + 徒手）
📌 特点：高频刺激（比五分化更适合自然训练）
📌 用法：直接照练（循环执行）

🧠 训练规则

  • 每个动作：3～4组 × 8～15次
  • 组间休息：60～90秒
  • 重量选择：最后2次吃力但能完成
  • 训练节奏：3天循环

🔥 热身（必做）

每次训练前 5–7 分钟

  • 肩部环绕：前后各20次
  • 猫牛式：10次
  • 开合跳：30秒 × 2组
  • 徒手深蹲：15次

📅 训练安排（三分化循环）

天数 内容
Day1 推（胸 + 肩前 + 三头）
Day2 拉（背 + 二头 + 后束）
Day3 腿 + 核心
Day4 休息 / 轻有氧
然后 循环

🔴 Day1：推（胸 + 肩 + 三头）

🟫 胸部

🏋️ 哑铃卧推

  • 肩胛收紧
  • 下放慢，推起快

🏋️ 哑铃窄推（夹胸推）


  • 哑铃始终夹紧
  • 推起时主动挤压胸肌
  • 控制速度

🏋️ 哑铃飞鸟（轻重量）

  • 手肘微弯
  • 控制拉伸

🟨 肩部

🏋️ 哑铃推肩

  • 核心收紧
  • 不锁死手肘

🏋️ 侧平举（递减组）

  • 小重量
  • 控制节奏

🟥 三头

🏋️ 哑铃颈后臂屈伸

  • 手肘固定
  • 不外扩

🔵 Day2：拉（背 + 二头 + 后束）

🟦 背部

🏋️ 跪姿单臂哑铃划船

  • 拉向髋部
  • 顶峰停顿

🏋️ 双手俯身划船

  • 背部打直
  • 拉向腹部

🏋️ 哑铃直臂下拉（模拟高位下拉）

  • 手臂微弯
  • 专注背阔发力

🟨 后束

🏋️ 俯身反向飞鸟

  • 手肘带动
  • 慢控制

🟥 二头

🏋️ 二头弯举

🏋️ 锤式弯举

🟢 Day3：腿 + 核心

🟩 腿部

🏋️ 哑铃深蹲

🏋️ 罗马尼亚硬拉

🏋️ 保加利亚分腿蹲

🟪 核心

🏋️ 抬腿卷腹（主力）

  • 12～20次 × 3组

🏋️ 仰卧抬腿

  • 控制下放

🏋️ 平板支撑

  • 30～60秒 × 3组

🔥 有氧安排（跳绳）

  • 推日：10–15分钟
  • 拉日：10–15分钟
  • 腿日：❌ 不跳
  • 休息日：快走或轻跳

🧘 拉伸（练后必做）

每个动作20–30秒：

  • 胸部拉伸
  • 背部拉伸
  • 大腿前侧
  • 大腿后侧
  • 肩部拉伸

🎯 总结

👉 三分化比五分化更适合减脂期
👉 高频训练 = 更容易保住肌肉
👉 背和腿优先级最高
👉 核心训练决定腹肌是否明显
👉 坚持 > 花哨动作
